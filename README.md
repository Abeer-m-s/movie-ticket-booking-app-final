# ANIMOV – Anime Movie Ticket Booking App

An anime-themed movie ticket booking web app with a dark, vibrant aesthetic.

## Features

- **Landing page**: Slideshow of anime movie posters with Login in navbar
- **Movies page**: Browse movies (Black Panther–style layout after login)
- **Theatre selection**: Choose theatre and show time (Book Now flow)
- **Seating**: Interactive seat selection
- **Payment**: Multiple payment options (card, UPI, net banking, cash)

## Tech Stack

- **Frontend**: React (Vite), React Router
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Auth**: JWT + bcryptjs

## Usage

1. Open ANIMOV
2. Use **Login** in the navbar, or **Get Started** on the landing page
3. Sign up if needed, then log in
4. Browse movies and click **Book Now**
5. Choose theatre and show time
6. Select seats and complete payment
